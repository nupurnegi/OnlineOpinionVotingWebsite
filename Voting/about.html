ABOUT

The aim of this project is to get users votes or choices on various fields such as food,sports,lifestyle,fashion. This would help various organisations who are more user directed to get information about what the users are interested in.For example the location where more people are interested in eating italian food would be the appropriate one for opening a pizza restaurant. This project would also help the casual users learn about what are the right things to use or implement with the help of the statistics provided. The voting management website would have databases for the information related to food,lifestyle,sports,fashion,users,the votes.There would be two logins:one for the casual user and other for the organisation. The data collected from the votes of users in various categories will be converted into statistics using rstudio and this will be used by the various organisations who are dependent upon the users choice.The vote or questions will be asked to the user on the categoriesFood,Lifestyle,Sports,Fashion. A casual user can login and vote and can also see what other people voted on the basis of the statistics calculated.Different questions will be asked to the user on different categories as mentioned, and their choices or votes will be recorded and used to calculate the statistics.The user would be able to vote on different categories on the basis of the information stored in the database. The front end would be implemented with javascript,html,css and the backend would be implemented with of php.The database would be created using Mysql.Xampp will be used as the server or the platform on which all the applications like mysql,php,filezilla etc would be worked upon.Once satisfied, a user can click a button to submit the vote poll. R studio will be used to calculate the statistics An organisation can login and view the statistics about the likes and dislikes of the people in a particular field.This would help the organisation to earn more profit since they they would be able to see what the users in an area is interested in by viewing the statistics calculated on the basis of the votes given by the users.The organisations cannot vote and they can only view the statistics on the basis of their interests.
